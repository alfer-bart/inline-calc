# inline-calc

// + = add \n
// - = subtract \n
// * = multiply \n
// / = divide \n
// x = last answer \n
// ^n = to the power of n \n
// rn = square root of n \n
// p = pi constant \n
// e = e constant \n
// (...) = expression in parentheses \n
// q = quit program \n

Example: \n
(15-(7+3))*2^3+r64 \n
outputs 48 \n
